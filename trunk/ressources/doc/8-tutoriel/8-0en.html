Sample loading

The sample preparation begins by loading .wav files, using the action "Import samples" in the "Files" menu. They may be obtained:

  • through Internet,
  • by recording a real musical instrument,
  • by synthesis using a specialized software.

Sample looping

Next, a loop may be required in the sample. This is the case for example of a flute sound: the length of the sample can be only one second but we would like to make it last indefinitely. Two positions, the start and end of the loop, has to be found in the sound wave so that locally the wave can be superimposed. Here is the method:

  • click in the tree on the sample to be looped,
  • in the editing page of samples, arbitrarily position the start and end of the loop in the graphic if no loop is already defined,
  • authorize looping and start playback,
  • adjust the start and / or end of the loop until the transition between the two positions is not noticeable.
If the looping process is tough, this tool may help.

Sample tuning

Finally, a tuning has to be done. To do this, a calibration tool is available in the editing page of samples. For each sample the method is as follows:

  • begin the playback (button "play"), if possible with loop,
  • add the sine calibration and adjust the volume to hear the two sounds as clearly as possible,
  • change the root key so that the two sounds best coincide,
  • adjust the correction to tune the sample with to root key (for this we must be attentive to the beating and make sure it is as slow as possible).
The frequency analysis can be a good indicator to start tuning.

To go further

Several tools are available to edit samples like:

  • removing blank at start,
  • equalizing the sound,
  • transposing,
  • normalizing the volume,
  • adjusting the balance for stereo samples.
