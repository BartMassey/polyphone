Pour créer une soundfont, cliquer sur « Nouveau » dans le menu « Fichier ». Les soundfonts étant composées d'une structure à trois niveaux, la création d'une soundfont passe ensuite par trois étapes fondamentales :

  • la préparation des échantillons,
  • la création d'instruments, contenant des échantillons,
  • la création de presets, contenant des instruments.

Une fois la soundfont créée, il est possible de jouer avec le synthétiseur intégré à Polyphone (à l'aide du clavier virtuel) ou d'utiliser un autre logiciel compatible sf2. Le format étant ancien est très répandu, les applications externes sont multiples. Voici quelques exemples :

  • jouer à l'aide de FluidSynth,
  • écoute de fichiers midi avec TiMidity, WildMIDI ou encore QuickTime,
  • éditer et lire des partitions avec MuseScore.
