What is a preset?

A preset is the entity visible from outside a soundfont and is identified by a bank number and a preset number. Each preset contains one or more instruments, such as instruments contain one or more samples.

Creating a preset and adding instruments

First of all, click on "New preset" in the toolbar if the icon is present, or in the contextual menu of the tree on one of the elements of the soundfont being edited. A name has to be filled for our new preset.

Then add instruments in the preset using a "drag & drop" in the tree: divisions appear in the preset.

It is possible to create as many presets as instruments, each preset containing only one instrument.

Setting up a preset

Changing the preset parameters may not be required since:

  • when creating a preset, bank number and preset number are automatically assigned,
  • when adding a instrument in a preset the key range is automatically calculated according to the instrument.
