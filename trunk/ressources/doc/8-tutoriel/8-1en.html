From this step, it is considered that one or more samples are available for the creation of an instrument.

Creating an instrument and adding samples

First of all, click on "New instrument" in the toolbar if the icon is present, or in the contextual menu of the tree on one of the elements of the soundfont being edited. A name has to be filled for our new instrument.

Then add samples in the instrument using a "drag & drop" in the tree: divisions appear in the instrument.

Note: when samples are created divisions are not copied but linked. There are no constraints on the number of divisions.

Disposition of divisions

Each division must then be positioned on the keyboard by changing the "Keyrange" in the table of the editing page of instruments. Generally, it is a good pratice to dimension the extent of the division with respect to the root key of the represented sample. The entire surface of the keyboard must be covered (eg, key 36 to key 96 for a classic synthesizer keyboard).

The tool "Automatic distribution" automatically distributes the samples over the keyboard.

Setting the divisions

First of all if the samples are looped the value "1" must be entered in the global division for the "Loop" parameter.

The global division, as its name suggests, allow you to enter a parameter for the entire instrument. A global parameter applies only to the divisions whose same parameter is not specified - ie the parameters of the classical divisions take precedence over the settings of the global division.

In the context of a simple tutorial no other parameters have to be edited for the instrument to be working. It is already possible to play with the virtual keyboard.

However, to improve the instrument it is advisable to enter a value in "Vol env release (s)", which prevents the sound to stop abruptly as soon as the key is released. The other parameters are described in the the description of tables.
