Afin de faciliter la recherche d'éléments dans l'arborescence, une barre de recherche est disponible pour effectuer un filtrage.

Lorsqu'une chaîne de caractères est insérée dans la barre de recherche :

  • si la chaîne de caractères est incluse dans le nom d'un instrument ou d'un preset, les instruments et presets restent affichés ainsi que leurs divisions,
  • si la chaîne de caractères est incluse dans le nom d'un échantillon, les échantillons concernés ainsi que tous les instruments contenant ces échantillons restent affichés.
Un clic sur la croix à droite du champ éditable annule le filtre.

L'arborescence peut par ailleurs être entièrement repliée ou dépliée grâce à deux boutons situés sur la gauche.
