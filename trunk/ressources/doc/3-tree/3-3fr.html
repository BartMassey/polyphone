Afin de faciliter la recherche d'éléments dans l'arborescence, une barre de recherche est disponible pour effectuer un filtrage.

Lorsqu'une chaîne de caractères est insérée dans la barre de recherche, les éléments la contenant sont affichés. De plus :

  • si la chaîne de caractères est incluse dans le nom d'un échantillon, les instruments utilisant l'échantillon et les presets utilisant les instruments utilisant le sample sont affichés,
  • si la chaîne de caractères est incluse dans le nom d'un instrument, les échantillons utilisés par l'instrument et les presets utilisant l'instrument sont affichés,
  • si la chaîne de caractères est incluse dans le nom d'un preset, les instruments utilisés par le preset et les échantillons utilisés par les instruments utilisés par le preset sont affichés.
Un clic sur la croix à droite du champ éditable annule le filtre.

L'arborescence peut par ailleurs être entièrement repliée ou dépliée grâce à deux boutons situés sur la gauche.
