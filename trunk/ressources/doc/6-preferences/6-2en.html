The colors of the graph in the sample page may be changed in the "Samples" tab. The button "reset" recalls the original parameters.

Moreover, it is possible to check the following options:

  • Wav file import: trim to loop
    When importing a wav file, data located after the end of loop are automatically removed (same effect than the tool "Trim to end of loop").
  • Wav file import: remove blank at start
    When importing a wav file, blank located before the beginning of the attack is automatically removed (same effect than the tool "Remove blank at start").
  • Stereo editing: change linked sample
    When editing a stereo sample, the 2nd channel is automatically updated. It is the same when editing an instrument division: the division corresponding to the second channel (with the same note and velocity ranges) is updated.
