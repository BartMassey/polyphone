PREFERENCES


Software preferences are accessible under "Edition" in the menu bar.

This dialog allows the modification of the following parameters:

  • Audio backend
    Specifies the audio backend used to play samples. Jack and Asio drivers (the latter for Windows only) may be chosen.
  • Midi input
    Specifies the midi input driving the virtual keyboard.
  • Importation of .wav files : trim to loop
    When importing a .wav file, data located next the end of loop are automatically removed.
  • Importation of .wav files : remove blank at start
    When importing a .wav file, blank located before the beginning of the attack is automatically removed.

This tab may be used to modify the chorus and reverb of the synth, along with its global volume.

The colors in the sample graph may be changed here. The button "reset" recalls the original parameters.

Most of the features present in the menus can be found in the toolbar by moving the corresponding action from the left list to the right list. The order of actions in the toolbar can be changed. The "reset" button recalls the original settings.
