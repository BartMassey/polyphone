PREFERENCES


Software preferences are accessible under "Edition" in the menu bar.

This dialog allows the modification of the following parameters:

  • Audio backend
    Specifies the audio backend used to play samples. Jack and Asio drivers (the latter for Windows only) may be chosen.
  • Midi input
    Specifies the midi input driving the virtual keyboard.
  • Importation of .wav files : trim to loop
    When importing a .wav file, data located next the end of loop are automaticaly removed.
  • Importation of .wav files : remove blank at start
    When importing a .wav file, blank located before the beginning of the attack is automaticaly removed.

This tab contains for now only one item modifying the synth volume (instrument and preset).
