MENU BAR


The menu bar contains the following elements:

  • File
    • New
      Create a new .sf2 file.
    • Open
      Open an existing .sf2 file.
    • Save
      Save the current file.
    • Save as
      Save the current file in another file.
    • Import
      Import samples in .wav format.
    • Export
      Export samples present in the .sf2 file in a .wav format.
    • Close file
      Close the current file.
    • Quit
      Quit the software.
  • Edition
    • Undo
      Undo the last action.
    • Redo
      Redo the last undone action.
    • Copy
      Copy an element being either in the tree or in a table.
    • Paste
      Paste a copied element.
    • Remove
      Remove the selected element, in the tree or a table.
    • Rename / Bulk rename
      Rename the selected element in the tree. Renaming several samples simultaneously is possible.
    • Preferences
      Open the software preferences.
  • Tools
    • Display the different tools available in function of the element selected in the tree.
  • View
    • Tools
      Display or hide the toolbar under the menu bar.
    • Modulators section
      Display or hide the section allowing the modulators edition.
    • Keyboard
      Modify the virtual keyboard length.
  • Help
    • Manual
      Show this manual.
    • About
      Show the software version.
