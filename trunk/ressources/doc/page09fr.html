Les préférences du logiciel sont accessibles en allant dans la barre de menu, "Édition", "Préférences".

Cette fenêtre de dialogue permet la modification des paramètres suivants :

  • Sortie audio
    Modifie la sortie audio utilisée lors de la lecture d'un son. Les drivers Jack et Asio (ce dernier pour Windows seulement) peuvent être choisis.
  • Taille du tampon
    Spécifie la taille du tampon utilisé par les drivers Jack et Asio. Un tampon de petite taille aura pour conséquence une faible latence, mais le son peut devenir de mauvaise qualité.
  • Entrée midi
    Modifie l'entrée midi pilotant le clavier virtuel.
  • Import fichiers wav : ajuster à la boucle
    Lors de l'import d'un fichier wav, les données situées après la fin de la boucle sont automatiquement enlevées.
  • Import fichiers wav : enlever le blanc au départ
    Lors de l'import d'un fichier wav, le blanc situé avant le début de l'attaque est automatiquement enlevé.

Cet onglet permet de paramétrer le chorus et la réverbération du synthétiseur, ainsi que son volume global. Le volume global peut être modifié via le controleur midi "volume".

Les couleurs du graphique des samples peuvent être modifiées ici. Le bouton "réinitialiser" rappelle les paramètres d'origine.

La plupart des fonctions présentes dans les menus peuvent se trouver dans la barre d'outils en déplaçant l'action correspondante de la liste de gauche vers la liste de droite. L'ordre des actions dans la barre d'outils peut être modifié. Le bouton "réinitialiser" rappelle les paramètres d'origine.

Les touches du clavier de l'ordinateur destinées à jouer des notes sont définies ici. Il est possible de monter ou descendre octave par octave les hauteurs de notes.
