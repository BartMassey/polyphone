The virtual keyboard allows us to test the current sample, instrument or preset. It can be controlled in three ways:

  • with the mouse by clicking on the desired key,
  • using the computer keyboard, the keys being defined in the preferences,
  • using an external keyboard, MIDI input being defined in the preferences.

To the left of the virtual keyboard are two values​​, representing the number and velocity of the note played.

To the right is a button offering two modes of use of the virtual keyboard according to its value:

  • if the know is turned to 0, the velocity used for reading is that received by the MIDI signal of an external keyboard,
  • if the button is not set to 0, the velocity used for reading is imposed by the value entered: the nuances that would be made via an external keyboard are therefore not taken into account.
