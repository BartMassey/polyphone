Le clavier virtuel permet d'écouter l'échantillon, l'instrument ou le preset en cours d'édition. Il est pilotable de trois manières différentes :

  • grâce à la souris en cliquant sur la touche souhaitée,
  • grâce au clavier de l'ordinateur, les touches étant définies dans les préférences,
  • à l'aide d'un clavier externe, l'entrée MIDI étant définie dans les préférences.

A gauche du clavier virtuel se trouvent le nom et la vélocité de la note jouée.

A droite se trouve un bouton proposant deux modes d'utilisation du clavier virtuel selon sa valeur :

  • si le bouton est réglé sur 0, la vélocité utilisée pour la lecture est celle reçue par le signal MIDI d'un clavier externe,
  • si le bouton n'est pas réglé sur 0, la vélocité utilisée pour la lecture est imposée par la valeur renseignée : les nuances qui seraient faites via un clavier externe ne sont donc pas prises en compte.
