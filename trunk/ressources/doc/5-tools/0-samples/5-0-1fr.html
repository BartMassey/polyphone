Le logiciel cherche une zone stable de l'échantillon pour ensuite créer une boucle artificiellement. Un segment du son est recopié avec un crossfade, de manière à ce que la transition à la jointure ne puisse pas s'entendre.

Un avertissement apparaît si aucune boucle ne peut être trouvée. Le cas se présente lorsque l'échantillon est trop court ou trop turbulent.

Attention : cet outil modifie l'échantillon original. Les modifications peuvent être annulées via la fonction « annuler », mais si le fichier est sauvegardé et si Polyphone est fermé il n'y a aucun moyen de revenir en arrière.
