Cet outil permet d'appliquer un filtre qui cassera toutes les fréquences dont l'intensité est au-dessus d'un seuil. Le seuil est défini par trois paramètres :

  • la fréquence à partir de laquelle le filtre commence à agir,
  • la fréquence à partir de laquelle le filtrage est complet,
  • la raideur qui définit l'évolution du seuil entre les deux précédentes fréquences (linéaire ou avec une chute brutale).
