Lorsqu'une sélection d'éléments ne se rapporte qu'à un seul fichier sf2 dans l'arborescence, le menu « Outils » propose les fonctionnalités suivantes :

  • Association auto samples
  • Dissocier les échantillons stéréo
  • Enlever les éléments non utilisés
  • Enlever tous les modulateurs
  • Exporter pics de fréquence (.csv)
  • Régler atténuation minimale
