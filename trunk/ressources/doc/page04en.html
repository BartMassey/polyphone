GLOBAL INFORMATION


When clicking on a header of a .sf2 tree (its name, "samples", "instruments" or "presets"), the page of global information edition appears.

It is possible to change the following information:

  • Name
    This is the internal name of the soundfont, which does not necessarily have to do with the file name. This name is used in the tree.
  • Author
    Relates to the name of the file author.
  • Copyright
    Mention to fill if the file is subject to copyright.
  • Date
    Creation date of the file.
  • Product
    Product for which is designed the soundfont.
  • Comments
    Information not included in any of the preceding categories.
  • 24 bits samples
    Indicates whether the samples should be recorded in 24 bit (16 bits otherwise).
Other information, not editable, can be found on the page:
  • File name
    Address and name of the file (different of the internal name of the soundfont).
  • Soundfont version
    Corresponds the version of the SF2 format used by the file.
  • Sound engine
    Corresponds to the name of a sound engine.
  • ROM name and version
    Name and version of a sample ROM if used.
  • Editing software
    Edition software used for the soundfont creation.
Last, a count is made on:
  • the number of samples,
  • the number of unused samples,
  • the number of instruments,
  • the number of unused instruments,
  • the number of presets,
  • the number of parameters edited for the instruments,
  • the number of parameters edited for the presets.
Note: the number of 65536 parameters in the instruments and the presets should not be exceeded. Some synthesizers may not be adapted to read them all.
