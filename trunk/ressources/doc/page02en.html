TOOLBAR


The toolbar contains shortcuts for some elements of the menu bar and of the contextual menu of the tree. The toolbar is also the place for the virtual keyboard.

From the left to the right:

  • New file
    Create a new .sf2 file.
  • Open file
    Open an existing .sf2 file.
  • Save file
    Save the current file.
  • Undo
    Undo the last modification.
  • Redo
    Redo the last undone action.
  • New sample
    Add new samples to the current file.
  • New instruments
    Create a new instrument.
  • New preset
    Create a new preset.
  • Remove element
    Remove one or more elements selected in the tree.
