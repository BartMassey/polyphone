The "File" menu contains the following elements:

  • New
    Create a new .sf2 file.
  • Open
    Open an existing .sf2 file.
  • Save
    Save the current file.
  • Save as
    Save the current file in another file.
  • Import samples
    Import .wav samples.
  • Import soundfont
    Import one or more sfz soundfont(s).
  • Export samples
    Export samples present in the .sf2 file in a .wav format.
  • Export (sfz)
    Export the current sf2 file in sfz or sf2, by selecting the different presets to be exported.
  • Close file
    Close the current file.
  • Quit
    Quit the software.
