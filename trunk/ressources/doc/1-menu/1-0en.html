The "File" menu contains the following elements:

  • New
    Create a new .sf2 file.
  • Open
    Open an .sf2, .sfz or .sfArk file.
  • Save
    Save the current file.
  • Save as
    Save the current file in another file.
  • Import samples
    Import .wav samples.
  • Export samples
    Export in a .wav format samples present in a soundfont.
  • Export (sfz)
    Export the current sf2 file(s) in sfz or sf2, by selecting the different presets to be exported.
  • Close file
    Close the current file.
  • Quit
    Quit the software.
