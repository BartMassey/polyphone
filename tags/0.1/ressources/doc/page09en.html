PREFERENCES


Software preferences are accessible under "Edition" in the menu bar.

This dialog allows the modification of the following parameters:

  • RAM management
    This parameter specifies the software behaviour during file opening (audio and .sf2) : it is possible to fully load all data or to only load the essential. In the first case the loading time may be longer, but following treatments will be faster.
  • Audio output
    Specifies the audio backend used during sample playing.
  • Importation of .wav files : trim to loop
    When importing a .wav file, data located next the end of loop are automaticaly removed.
  • Importation of .wav files : remove blank at start
    When importing a .wav file, blank located before the beginning of the attack is automaticaly removed.
Note : restarting the software will be needed to take some of the parameters into account.
